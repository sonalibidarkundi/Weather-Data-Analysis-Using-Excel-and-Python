# Weather Data Analysis using Python

## 📊 Project Overview

This project analyzes weather data of different cities using Python libraries like pandas, numpy, and matplotlib.

## 🎯 Features

* Data cleaning and analysis
* Finding hottest and coldest cities
* Average temperature, humidity, rainfall
* Data visualization (bar chart, pie chart, histogram)

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib

## 📁 Dataset

The dataset contains city-wise weather details such as temperature, humidity, and rainfall.

## 📈 Output

* Identified climate patterns
* Visualized data using graphs

## ▶️ How to Run

1. Install required libraries:
   pip install pandas numpy matplotlib

2. Run the script:
   python weather_analysis.py
