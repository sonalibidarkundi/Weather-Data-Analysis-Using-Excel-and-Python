# 📊 Weather Data Analysis using Excel & Python

## 📌 Project Overview

This project focuses on analyzing weather data of different cities using both **Microsoft Excel** and **Python**. The goal is to extract meaningful insights such as temperature trends, humidity distribution, and rainfall patterns.

---

## 🎯 Objectives

* Perform data cleaning and preprocessing
* Analyze weather parameters
* Visualize data using charts and graphs
* Compare analysis using Excel and Python

---

## 🛠️ Tools & Technologies Used

* **Python**

  * pandas
  * numpy
  * matplotlib
* **Microsoft Excel**

  * Data cleaning
  * Sorting & filtering
  * Charts (bar, pie)

---

## 📁 Dataset

The dataset contains the following columns:

* City
* Temperature
* Humidity
* Rainfall

---

## 🔍 Work Done

### 📊 In Excel

* Cleaned the dataset (handled missing values)
* Sorted and filtered data
* Created basic charts:

  * Bar chart (Temperature by City)
  * Pie chart (Humidity distribution)

### 🐍 In Python

* Loaded dataset using pandas
* Performed data analysis:

  * Identified hottest and coldest cities
  * Calculated average temperature, humidity, and rainfall
* Data visualization using matplotlib:

  * Bar chart
  * Pie chart
  * Line chart
  * Histogram

---

## 📈 Key Insights

* Identified cities with highest and lowest temperatures
* Observed rainfall trends across cities
* Compared humidity distribution
* Visualized overall weather patterns

---

## ▶️ How to Run (Python)

1. Install required libraries:

   ```
   pip install pandas numpy matplotlib
   ```

2. Run the script:

   ```
   python weather_analysis.py
   ```

---

## 📸 Output

* Graphs showing temperature, rainfall, and humidity
* Printed analysis results in console

## 🚀 Conclusion

This project demonstrates how both Excel and Python can be used together for effective data analysis and visualization. Python provides more flexibility and automation, while Excel helps in quick data inspection and basic analysis.
